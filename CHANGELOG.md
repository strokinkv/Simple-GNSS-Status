# Changelog

## 1.0.0

Первый стабильный релиз Simple GNSS Status.

### Возможности

- Отображение координат, если Android уже вычислил местоположение.
- Отображение точности координат.
- Счетчик спутников `используется / видно`.
- Средний уровень сигнала только по спутникам, которые используются в расчете координат.
- Таблица спутников с ID, флагом системы, признаком использования и уровнем сигнала.
- Скрытие спутников с нулевым уровнем сигнала.
- Темная минималистичная тема.

### Ограничения

- Для реального GNSS-статуса нужен физический Android-телефон.
- Приложение не управляет A-GPS/XTRA и не показывает A-GPS-статус.
- APK распространяется через GitHub Releases и устанавливается вручную.
